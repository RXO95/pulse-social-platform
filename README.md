# Pulse – Social Media Backend API

A Twitter / Reddit-style backend built with FastAPI and MongoDB.

## Features
- User authentication (JWT)
- Secure password hashing (Argon2)
- Create posts
- Timeline feed
- Like system
- MongoDB Atlas integration

## Tech Stack
- FastAPI
- MongoDB (Motor)
- JWT Authentication
- Argon2
- Python

## API Modules
- /auth → signup & login
- /posts → create post
- /feed → timeline
- /likes → like posts

## Status
Phase 1 and Phase 2 core backend completed.
